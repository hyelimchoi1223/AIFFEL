## Git과 GitHub에 대해서
---
### Git vs GitHub
* Git : 소스코드 버전 관리 시스템
* GitHub : Git으로 관리되는 버전 관리를 호스팅해 다른 사람들과 협업할 수 있도록 도와주는 온라인 서비스

### Git 사용법
#### Git 생성 및 커밋
* 저장소 생성: 해당 폴더에 새로운 저장소를 생성
> ```bash
> git init
> ```
* 새로운 파일을 인덱스에 기록
> ```bash
> git add [filename]
> ```
